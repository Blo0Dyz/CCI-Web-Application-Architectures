#### Margin

``` Margin: ``` is the <strong>outer</strong> spacing around an element ( Creating distance between elements ).

In CSS Margin can be set as <strong>Individual</strong>:
```
    p{
        margin-top: 100px;
        margin-right: 100px;
        margin-bottom: 50px;
        margin-left: 50px;
    }
```



``` Padding:``` is the <strong>inner</strong> spacing inside an element ( Creating space between the content and the border ).
